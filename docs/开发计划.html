NFC时间管控器

基于ESP32-S3的Nintendo Switch使用时间管理系统

V1.2

项目概述

ESP32-S3 时间管控器，用于管理 Nintendo Switch 游戏机使用时间。核心功能是检测NFC标签，判断Switch是否在盒内，超时报警。

已完成功能 已完成

WiFi配网模块

完整的WiFi配网功能，支持AP模式和自动重连。

  • 上电无WiFi配置时自动进入配网模式
  • 连接失败3次后自动进入配网模式
  • 配网热点: TimeControl (密码: 12345678)
  • 配网页面支持WiFi热点自动扫描
  • 支持手动输入隐藏SSID

NFC验证功能

基于UID的NFC标签验证，防止非授权设备使用。

  • NVS中未注册UID时，不做校验
  • NVS中有注册UID时，验证NFC UID是否匹配
  • UID匹配: 亮绿灯
  • UID不匹配或NFC离开: 亮红灯

NFC离开计时功能

精确记录Switch离开时长，支持断电续算。

  • NFC离开时开始计时
  • NFC在位且UID匹配时停止计时
  • NFC在位但UID不匹配时继续计时
  • 断电后当天可继续累计
  • 跨天自动重置计时器

超时报警功能

离开超时后发出紧迫的声光报警。

  • 超时阈值: 30分钟
  • 红灯快速闪烁 (200ms间隔)
  • 蜂鸣器急促报警 (滴滴滴-滴滴滴)
  • 正确的NFC UID放回后停止报警

按键功能

GPIO0按键支持配网和注册功能。

  • 长按3秒: 进入配网模式
  • 短按 + NFC在位: 保存当前NFC UID到NVS
  • 使用中断方式监听，释放时处理

NTP网络对时

WiFi连接后自动同步网络时间。

  • 使用阿里云NTP服务器
  • 用于跨天判断

硬件接线

ESP32引脚 功能 备注
GPIO1 NFC SDA I2C数据
GPIO2 NFC SCL I2C时钟
GPIO4 NFC IRQ 中断信号
GPIO12 LED 红色 串联220Ω电阻
GPIO13 LED 绿色 串联220Ω电阻
GPIO14 蜂鸣器 有源5V，低电平触发
GPIO0 配置按钮 内部上拉，长按3秒触发

LED状态说明

状态 LED行为
配网模式 红绿灯交替闪烁 (500ms)
NFC在位 + UID匹配 绿色常亮
NFC离开或UID不匹配 红色常亮
超时报警 红色快速闪烁 (200ms)
